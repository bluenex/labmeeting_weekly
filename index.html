Effect of Virtual Reality on
Cognition in Stroke Patients

Week 1

Introduction

The affected area of cognitive function

  • Can be only one area of cognitive function
  • But in most cases, it affects all area including:

Attention, Concentration, Memory, Spatial processing skills,
Language, Problem-solving skills, Planning skills

Cognitive impairment after
stroke can affects:

  • Independent function
  • Activity participation
  • Decreasing in morale and motivation
    to participate in rehabilitation

Most studies use virtual reality in cognitive
function to assess unilateral neglect or overall
cognitive impairment


Whereas only few studies proved effectiveness
of virtual reality on cognitive impairment

Materials and Methods

Study subjects

  • First stroke
  • 28 patients
  • Already pass acute phase treatment
  • K-MMSE score range from 10-24
  • Virtual reality & Control groups

Research methods

VR group

  • VR training 30 min, 3 times/week
  • Computer based training 30 min, 2 times/week
  • For 4 weeks

Control group

  • Computer based training 30 min, 5 times/week
  • For 4 weeks

Virtual reality training

  • Use commercial product called IREX system®
  • 5 out of 20 games were chosen
    • Bird and Balls
    • Coconuts
    • Conveyer
    • Juggler
    • Soccer
  • For each times, 4 games will be used
  • 6 minutes training with 2 minutes
    break for each game

Virtual reality experimental set up of IREX System®

Computer-assisted cognitive training

  • Use commercial product called ComCog®
  • Designed to promote attention and memory
    for patients with cognitive impairment
  • Both attention and memory program consist of three
    courses (beginner, intermediate and advanced)

Training program in details

Attention

  • basic visual perception training
  • auditory perception training
  • attention training
  • attention discrimination training
  • continuous attention training
  • attention integration training
  • emotion training

Memory

  • simple recognition memory training
  • simple spatial memory training
  • sequential recall memory training
  • sequential verbal recall memory training
  • associated recall memory training
  • verbal categorization memory training
  • integrated memory training

Evaluation methods

Before & After treament

  • Computerized Neuropsychological
    Test (CNT, MaxMedica)
  • Tower of London test (TOL)
  • Korean-Modified Barthel Index (K-MBI)
  • Motricity Index (MI)

CNT in details

  • Visual & Auditory continuous performance tests (CPT)
  • Word-Color test
  • Forward & Backward digit span tests (DST)
  • Verbal learning test for verbal memory evaluation tests
  • Spatial memory test
